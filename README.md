# transportation-research-writing

Tools and reference material for rewriting, drafting, and polishing transportation, operations research, and AI-for-transportation research writing.

## What this repo includes

- `SKILL.md`: workflow and output rules for this writing assistant
- `references/`: detailed guidance for sentence revision, paper text, and rebuttal writing
- `agents/`: agent configuration used by the repo

## Typical use

- Rewrite rough Chinese or English sentences into publication-ready academic English
- Draft manuscript paragraphs with clear logic and standard academic tone
- Prepare concise, point-by-point reviewer responses
